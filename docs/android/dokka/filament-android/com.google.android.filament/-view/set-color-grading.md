# setColorGrading

[main]\
open fun [setColorGrading](set-color-grading.md)(colorGrading: [ColorGrading](../-color-grading/index.md))

Sets this View's color grading transforms.

#### Parameters

main

| | |
|---|---|
| colorGrading | Associate the specified [ColorGrading](../-color-grading/index.md) to this view. A ColorGrading can be associated to several View instances. Can be null to dissociate the currently set ColorGrading from this View. Doing so will revert to the use of the default color grading transforms. |
