# Filmic

[main]\
open class [Filmic](index.md) : [ToneMapper](../index.md)

&quot;Filmic&quot; tone mapping operator. This tone mapper was designed to approximate the aesthetics of the ACES RRT + ODT for Rec.709 and historically Filament's default tone mapping operator. It exists only for backward compatibility purposes and is not otherwise recommended.
