# PREMULTIPLIED_SRGB

[main]\
[PREMULTIPLIED_SRGB](index.md)

The color is defined in sRGB space and the RGB values have been premultiplied by the alpha (for instance, a 50% transparent red is <0.5,0,0,0.5>).
